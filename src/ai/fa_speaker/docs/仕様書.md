# fa_speaker 仕様書

## 1. 目的

`fa_speaker` は `AudioFrame` から speaker embedding、speaker id、verification score を生成する AI node である。
ASR transcript、speaker diarization timeline、dialogue policy は責務外とする。

## 2. 入力/出力

入力は config と model capability が一致する canonical `AudioFrame` とする。
supported input は `FLOAT32LE`、`bit_depth=32`、mono、interleaved、configured sample rate、configured window に限定する。
`PCM16`、`PCM32`、stereo、planar、sample-rate mismatch、window mismatch、non-finite sample、range violation は変換せず fail closed または frame reject とする。

出力は speaker embedding / id / verification score、model id、source identity、stream identity を持つ明示 message / payload とする。

## 3. Model Capability

`fa_speaker` は package 化前の roadmap note であり、以下は package 化時に確定する capability contract である。

- model identity: model id、必要な場合のみ model path
- provider: backend name
- worker command または endpoint
- supported sample rates
- supported window
- execution mode: batch speaker embedding または streaming speaker verification
- output schema: speaker embedding、speaker id、verification score、model id、source identity、stream identity
- embedding dimension
- enrolled speaker index

## 4. 責務外

- ASR
- diarization timeline segmentation
- dialogue policy
- resample / channel conversion / gain
- unknown speaker の勝手な既知 speaker 割り当て

## 5. fail closed 条件

- `backend.name` 欠損
- model path / model id 欠損
- provider / worker command 欠損
- supported sample rates / window 欠損
- execution mode 欠損
- enrolled speaker index 欠損
- embedding dimension 欠損または mismatch
- input metadata mismatch
- backend output が finite score/vector でない
