# fa_vad テスト設計

## Unit Test

- `FA-VAD-SPEC-001`: backend.name 空で `RuntimeError`。
- `FA-VAD-SPEC-002`: unknown backend で `RuntimeError`。
- `FA-VAD-SPEC-003`: `channels != 1` を reject し、mono downmix しない。
- `FA-VAD-SPEC-004`: unsupported bit depth を reject する。
- `FA-VAD-SPEC-005`: `backend.model_path` missing / not found で `RuntimeError`。
- `FA-VAD-SPEC-006`: `backend.execution_provider` missing / unknown で `RuntimeError`。
- `FA-VAD-SPEC-007`: `sample_rate != target_sample_rate` を reject し、node 内 resample しない。
- `FA-VAD-SPEC-008`: normalized range 外または non-finite samples を reject する。
- `FA-VAD-SPEC-009`: `backend.command` missing / not executable で `RuntimeError`。
- `FA-VAD-SPEC-010`: `backend.args` に `{audio}`, `{model}`, `{provider}`, `{sample_rate}` がなければ `RuntimeError`。
- `FA-VAD-SPEC-011`: backend module が `torch` を import せず、external process boundary を使う。
- `FA-VAD-SPEC-012`: fake external worker で WAV / stdout probability contract を検証する。
- `FA-VAD-SPEC-013`: `qos.depth <= 0` を `max(1, depth)` などで補正せず `RuntimeError`。
- `FA-VAD-SPEC-014`: invalid threshold / hangover / sample rate を backend 側でも `RuntimeError`。
- `FA-VAD-SPEC-015`: backend constructor は sample rate / frame / threshold / hangover の暗黙 default を持たない。
- `FA-VAD-SPEC-016`: empty `AudioFrame.data` を silent skip せず invalid frame として reject する。
- `FA-VAD-SPEC-017`: backend runtime failure で fatal log / shutdown / exception により fail closed する。

## Integration Test

- fixture PCM frame を流し、`VadState` が publish されることを確認する。
- fake external worker で speech start / end の backend pipeline を確認する。

## Launch Test

- `fa_vad.launch.py` が config file を読み込むことを確認する。
- 明示 model path / execution provider がない config で起動失敗することを確認する。
- default config が external worker command を空にしており、明示設定なしで成功扱いしないことを確認する。

## Colcon Test

- `CMakeLists.txt` で `ament_cmake_pytest` に `test/` を登録し、`colcon test --packages-select fa_vad` で unit / integration / launch contract test を実行する。
- test 実行時は ambient pytest plugin を無効化し、host/container 側の無関係な pytest plugin によって VAD 契約テストが skip されないことを保証する。

## Fixtures

- silence mono float32
- speech-like mono float32
- malformed frame
- `test/fixtures/fake_vad_worker.py`: external worker contract 用に stdout probability を返す mock。
