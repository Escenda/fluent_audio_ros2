# fa_audio_embedding 仕様書

## 1. 目的

`fa_audio_embedding` は `AudioFrame` または明示 feature payload から audio embedding を生成する AI node である。
embedding model runtime は `backends/` 配下に分離し、ROS2 topic/message を知らない。

## 2. 入力/出力

入力は config と一致する `AudioFrame` または feature stream とする。
出力は embedding vector、model id、embedding dimension、source identity、stream identity を持つ明示 message / payload とする。

## 3. 責務外

- ASR / VAD / KWS
- speaker verification
- audio generation
- resample / channel conversion / gain
- missing model fallback

## 4. fail closed 条件

- `backend.name` 欠損
- model path / model id 欠損
- embedding dimension mismatch
- input metadata mismatch
- backend output が finite vector でない
