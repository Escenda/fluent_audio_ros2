# fa_audio_embedding アルゴリズム詳細説明書

1. startup config で `backend.name`、model path/model id、expected input format、embedding dimension を検証する。
2. input frame の source identity、stream identity、format metadata を検証する。
3. backend に canonical tensor を渡す。
4. backend output の dimension、dtype、finite value contract を検証する。
5. 契約を満たす場合だけ embedding result を publish する。

embedding 失敗時に zero vector や stale embedding を返さない。
Python version / venv が異なる backend は external worker / process / container 境界にする。
