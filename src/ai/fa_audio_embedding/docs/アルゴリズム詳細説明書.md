# fa_audio_embedding アルゴリズム詳細説明書

1. startup config で `backend.name`、`backend.command`、`backend.model_id`、`backend.args`、expected input identity、expected input format、embedding dimension を検証する。
2. input frame の source identity、stream identity、format metadata を検証する。
3. `AudioFrame.data` を raw `float32le` として decode し、finite / normalized range を検証する。
4. backend に canonical mono `float32` vector と sample rate / source identity / stream identity を渡す。
5. backend output の dimension、dtype、finite value contract を検証する。
6. 契約を満たす場合だけ `AudioEmbeddingFrame` を publish する。

embedding 失敗時に zero vector や stale embedding を返さない。
Python version / venv が異なる backend は external worker / process / container 境界にする。
