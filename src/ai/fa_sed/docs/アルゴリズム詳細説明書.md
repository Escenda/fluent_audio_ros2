# fa_sed アルゴリズム詳細説明書

1. startup config で `backend.name`、model path/model id、label set、expected input format を検証する。
2. input frame の source identity、stream identity、format metadata を検証する。
3. backend に canonical audio tensor を渡す。
4. backend output の label、confidence、time range を検証する。
5. 契約を満たす場合だけ event result を publish する。

unknown label は default その他に丸めず fail closed とする。
Python version / venv が異なる backend は external worker / process / container 境界にする。
