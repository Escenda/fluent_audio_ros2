# fa_sed 仕様書

## 1. 目的

`fa_sed` は `AudioFrame` から sound event labels と confidence を生成する AI node である。
VAD / KWS / ASR とは独立した event detection contract とする。

## 2. 入力/出力

入力は config と model capability が一致する canonical `AudioFrame` とする。
supported input は `FLOAT32LE`、`bit_depth=32`、mono、interleaved、configured sample rate、configured window に限定する。
`PCM16`、`PCM32`、stereo、planar、sample-rate mismatch、window mismatch、non-finite sample、range violation は変換せず fail closed または frame reject とする。

出力は event label、confidence、time range、model id、source identity、stream identity を持つ明示 message / payload とする。

## 3. Model Capability

`fa_sed` は package 化前の roadmap note であり、以下は package 化時に確定する capability contract である。

- model identity: model id、必要な場合のみ model path
- provider: backend name
- worker command または endpoint
- supported sample rates
- supported window
- execution mode: batch detection または streaming detection
- output schema: event label、confidence、time range、model id、source identity、stream identity
- label set

## 4. 責務外

- VAD / KWS / ASR
- speaker recognition
- audio generation
- resample / channel conversion / gain
- unknown label set の自動推測

## 5. fail closed 条件

- `backend.name` 欠損
- model path / model id 欠損
- provider / worker command 欠損
- supported sample rates / window 欠損
- execution mode 欠損
- label set 欠損
- input metadata mismatch
- backend output が configured label set と一致しない
