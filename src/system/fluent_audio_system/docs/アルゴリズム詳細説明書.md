# fluent_audio_system アルゴリズム詳細説明書

1. `config` launch argument を解決する。
2. `${share:package}` を含む config path を展開し、YAML file の存在を確認する。
3. YAML root, `system`, `groups`, `nodes` の型を検証する。
4. disabled group / node を skip する。
5. enabled node の `package`, `exec`, `node_name`, `namespace`, `params_file`, `parameters`, `remappings` を検証する。
6. `${share:package}` を含む `params_file` を展開し、指定されている場合は存在確認する。
7. `params_file` と inline `parameters` を `launch_ros.actions.Node.parameters` に渡す。
8. `remappings` を `launch_ros.actions.Node.remappings` に渡す。
9. `TimerAction` で node 起動間隔を付ける。

YAML を書き換えたり、一時 params file を生成したりしません。
