# fa_network_in 仕様書

## 1. 目的

`fa_network_in` は明示 endpoint から audio payload を受け取り、configured `AudioFrame` contract として publish する source adapter である。
network jitter absorption、PLC、clock drift correction は `src/streaming` の責務とする。

## 2. 責務外

- jitter buffer / PLC / clock drift correction
- codec decode
- resample / channel conversion / gain
- endpoint 推測
- missing endpoint fallback

## 3. fail closed 条件

- endpoint 欠損
- backend 未指定
- expected format 欠損
- packet payload が expected format と一致しない
- source identity 欠損
