# fa_network_in アルゴリズム詳細説明書

1. startup config で endpoint、backend、expected format、source identity を検証する。
2. backend で network endpoint を開き、audio packet を受け取る。
3. packet payload と metadata を expected contract と照合する。
4. `AudioFrame` を publish する。

transport instability はこの node で補正しない。必要な場合は `src/streaming` の node を pipeline に明示する。
