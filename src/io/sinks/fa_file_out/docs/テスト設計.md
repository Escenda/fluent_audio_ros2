# fa_file_out テスト設計

## Unit

- 標準資料が存在すること。
- roadmap-only package として `package.xml` / `CMakeLists.txt` を持たないこと。
- backend contract document が encoding、resample、gain の暗黙処理を禁止していること。

## Integration

- fake writer backend で expected format と一致する frame だけを書き出すこと。
- expected format 不一致で起動または処理を fail closed すること。
- overwrite disabled の既存 target を fail closed すること。

## Launch

- `backend.name` 欠損で起動失敗すること。
- `file.path` 欠損で起動失敗すること。
- encoded output を要求する config が `fa_encode` stage なしで通らないこと。
