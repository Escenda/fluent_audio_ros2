# fa_file_out 仕様書

## 目的

`fa_file_out` は、`AudioFrame` を raw PCM file に書き出す将来の standalone
sink adapter の設計枠である。

## 現在の状態

現時点では ROS 2 package ではない。`package.xml`、`CMakeLists.txt`、launch
entrypoint を持たない。

## 非目標

- WAV / MP3 / Opus / FLAC encode を隠すこと
- resample、channel convert、bit depth convert、limiter、normalize、fade を行うこと
- `fa_out` の `pcm_file_writer` backend の互換 alias になること

## 現行契約

- system config は `package: fa_file_out` を参照しない。
- raw PCM file sink は `fa_out` の `backend.name: pcm_file_writer` で明示する。
- codec encode が必要な場合は `fa_encode` を pipeline に明示的に置く。

## Fail Closed

standalone package 化するまでは、`fa_file_out` を build / launch 対象として扱ってはならない。
