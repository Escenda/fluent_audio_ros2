# fa_network_out テスト設計

## Unit

- 標準資料が存在すること。
- roadmap-only package として `package.xml` / `CMakeLists.txt` を持たないこと。
- backend contract document が jitter、PLC、codec encode、endpoint guessing の暗黙処理を禁止していること。

## Integration

- fake sender backend で expected format と一致する frame だけを送信すること。
- expected format 不一致で起動または処理を fail closed すること。
- send failure を成功扱いしないこと。

## Launch

- `backend.name` 欠損で起動失敗すること。
- `endpoint.uri` 欠損で起動失敗すること。
- streaming stabilization をこの adapter config へ混ぜた場合に contract error になること。
