# fa_network_out 仕様書

## 目的

`fa_network_out` は、`AudioFrame` を network endpoint へ raw PCM packet として
送信する将来の standalone sink adapter の設計枠である。

## 現在の状態

現時点では ROS 2 package ではない。`package.xml`、`CMakeLists.txt`、launch
entrypoint を持たない。

## 非目標

- jitter buffer、packet loss concealment、clock drift correction を隠すこと
- network codec encode を隠すこと
- resample、limiter、normalize、fade を行うこと
- `fa_out` の `network_pcm_sender` backend の互換 alias になること

## 現行契約

- system config は `package: fa_network_out` を参照しない。
- raw PCM UDP sink は `fa_out` の `backend.name: network_pcm_sender` で明示する。
- network transport stabilization が必要な場合は `src/streaming` の専用 node を pipeline に置く。

## Fail Closed

standalone package 化するまでは、`fa_network_out` を build / launch 対象として扱ってはならない。
