# Frequency Processing

This category contains nodes that shape frequency content.

Examples:

- EQ
- low-pass filter
- high-pass filter
- band-pass filter
- notch filter
- de-esser
- spectral subtraction
- Wiener filter

Frequency processing can be useful before VAD, KWS, ASR, speaker embedding, or
playback, but it must be represented as explicit pipeline nodes.
