# fa_filter

Roadmap directory for the frequency wrapper node that composes explicit filter
stages such as high-pass, low-pass, band-pass, notch, EQ, de-esser, spectral
subtraction, and Wiener filtering.

This is not a ROS 2 package yet. Do not add `package.xml` until the package has
its specification, algorithm notes, backend documentation, launch contract, and
tests.
