# fa_wiener 仕様書

## 1. 目的

`fa_wiener` は `FLOAT32LE` interleaved `AudioFrame` に対して STFT based Wiener filter を適用する frequency processing node である。
signal/noise estimate は config と runtime estimator contract で明示し、欠損時に pass-through しない。

## 2. 入力

- message: `fa_interfaces/msg/AudioFrame`
- encoding: `FLOAT32LE`
- bit depth: `32`
- layout: `interleaved`
- sample rate / channels: startup config と一致

## 3. 出力

出力は入力と同じ sample rate、channels、encoding、bit depth、layout を保持する。
`source_id` は保持し、`stream_id` は `output_topic` に更新する。

## 4. 責務外

- device I/O
- resample / channel conversion / bit-depth conversion
- gain / normalize / compressor
- VAD / ASR / embedding
- missing noise estimate の自動補完
- pass-through degraded mode

## 5. fail closed 条件

- backend 未指定
- frame size / hop size / window 不正
- noise estimate config 欠損
- input frame metadata mismatch
- non-finite sample
- output が finite でない
