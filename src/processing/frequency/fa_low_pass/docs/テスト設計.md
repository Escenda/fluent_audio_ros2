# fa_low_pass テスト設計

## Scope

`test/unit/test_fa_low_pass_audio_frame_contract.py` は package の静的 contract を検査する。
ROS graph の起動や実 audio device は扱わない。

## 検査項目

- default config が `FLOAT32LE` / 32-bit / interleaved / 16 kHz / 1 ch を要求する。
- `filter.cutoff_hz` の既定値が `3400.0` で Nyquist 未満である。
- device I/O、resampling、format conversion、gain、normalize、limiter、denoise を持たない。
- 起動時 parameter validation が fail closed である。
- runtime frame validation が source/stream/format/data size を検査する。
- `stream_id` を `output_topic` へ更新し、それ以外の identity を保持する。
- low-pass coefficient と recurrence が仕様式に一致する。
- `source_id` 変更時に state を reset する。
- diagnostics が cutoff、alpha、state source、counter を含む。
- package layout と colcon pytest registration が標準形に一致する。

## 実行

```bash
env PYTHONDONTWRITEBYTECODE=1 PYTEST_DISABLE_PLUGIN_AUTOLOAD=1 python3 -m pytest -p no:cacheprovider src/processing/frequency/fa_low_pass/test/unit -q
```
