# fa_spectral_subtraction 仕様書

## 1. 目的

`fa_spectral_subtraction` は `FLOAT32LE` interleaved `AudioFrame` に対して STFT based spectral subtraction を適用する frequency processing node である。
noise profile は config で明示するか、明示的な calibration input から作成する。

## 2. 入力

- `input_topic`: ROS 搬送路
- `input_stream_id`: 受理する入力 `AudioFrame.stream_id`
- message: `fa_interfaces/msg/AudioFrame`
- encoding: `FLOAT32LE`
- bit depth: `32`
- layout: `interleaved`
- sample rate / channels: startup config と一致

## 3. 出力

- `output_topic`: ROS 搬送路
- `output.stream_id`: 出力 `AudioFrame.stream_id`

出力は入力と同じ sample rate、channels、encoding、bit depth、layout を保持する。
`source_id` は保持し、`stream_id` は `output.stream_id` に更新する。

## 4. 責務外

- device I/O
- resample / channel conversion / bit-depth conversion
- gain / normalize / compressor
- VAD / ASR / embedding
- missing noise profile の自動推定
- noise profile 欠損時の pass-through

## 5. fail closed 条件

- backend 未指定
- frame size / hop size / window 不正
- noise profile 欠損
- input frame metadata mismatch
- topic と stream identity の衝突
- non-finite sample
- output が finite でない
