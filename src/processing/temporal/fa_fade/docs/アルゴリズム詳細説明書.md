# fa_fade アルゴリズム詳細説明書

## 1. 入力モデル

入力は FLOAT32LE interleaved `AudioFrame` である。1 フレームのバイト数は `channels * sizeof(float)` とする。

## 2. 位置カウンタ

ノードは accepted frame のみで進む `position_frames_` を保持する。初期値は `fade.initial_position_frames` である。フレームが drop された場合、位置は進めない。

## 3. gain 計算

`duration = fade.duration_frames`、`position = position_frames_ + frame_index` とする。

```text
fade_in:  gain = min(1, position / duration)
fade_out: gain = max(0, 1 - position / duration)
```

計算は `double` で行う。同じ音声フレーム内の全 channel は同じ frame position の gain を共有する。

## 4. サンプル処理

1. `float` サンプルを `data` から読み出す。
2. サンプルが有限で `[-1.0, 1.0]` の範囲内であることを確認する。
3. `double faded = sample * gain` を計算する。
4. `faded` が有限で `[-1.0, 1.0]` の範囲内であることを確認する。
5. `float` に戻して出力 `data` へ書き込む。

範囲外の値は修正しない。clamp や normalize ではなく、フレーム単位で drop する。

## 5. 状態更新

全サンプルが正常に処理できた場合のみ、`position_frames_ += frame_count` とする。処理途中で不正値を検出した場合は publish せず、位置も進めない。
