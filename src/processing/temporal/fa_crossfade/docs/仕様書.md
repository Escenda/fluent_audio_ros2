# fa_crossfade 仕様書

## 1. 目的

`fa_crossfade` は2つ以上の audio segment を、明示的な overlap 長と fade curve に従って滑らかに接続する temporal processing node である。

## 2. 入力

入力は `fa_interfaces/msg/AudioFrame` または segment boundary を持つ stream とする。
各 segment は source identity、stream identity、sample rate、channels、encoding、bit depth、layout が一致する必要がある。

## 3. 出力

出力は入力と同じ sample rate、channels、encoding、bit depth、layout を保持する。
`stream_id` は `output_topic` に更新する。

## 4. 責務外

- device / file / network I/O
- resample / channel conversion / bit-depth conversion
- gain normalize / limiter
- routing mixer / bus routing
- missing segment の補完
- segment boundary 欠損時の推測 crossfade

## 5. fail closed 条件

- overlap length が未指定または不正
- fade curve が unsupported
- adjacent segment の format metadata mismatch
- segment boundary 欠損
- non-finite sample
- output が normalized range を超える
