# fa_pitch_shift 仕様書

## 1. 目的

`fa_pitch_shift` は `FLOAT32LE` interleaved `AudioFrame` の duration を保ったまま pitch を semitone / ratio 指定で変更する temporal processing node である。

## 2. 入力

- message: `fa_interfaces/msg/AudioFrame`
- encoding: `FLOAT32LE`
- bit depth: `32`
- layout: `interleaved`
- sample rate / channels: startup config と一致

## 3. 出力

出力は入力と同じ sample rate、channels、encoding、bit depth、layout を保持する。
duration contract は入力 frame count と同じである。

## 4. 責務外

- device / file / network I/O
- resample
- time stretching without pitch change
- voice conversion
- VAD / ASR / embedding
- pitch estimate の自動抽出
- unsupported frame length の padding / truncate

## 5. fail closed 条件

- shift ratio / semitone が未指定または不正
- frame length / hop length / window 不正
- input frame metadata mismatch
- non-finite sample
- output frame count が入力と一致しない
- output が finite/normalized contract を満たさない
