# fa_window テスト設計

## Unit Contract Tests

`test/unit/test_fa_window_audio_frame_contract.py` は、実行環境に ROS2 node を起動せず、config と source を検査して package contract を固定する。

確認項目:

- default config が FLOAT32LE interleaved 契約を表す
- window が device I/O、resampling、format conversion、channel conversion、gain、limiter、filter、denoise を持たない
- 起動時 validation が fail closed になっている
- runtime frame validation が source/stream identity、format、data size、frame count を確認する
- strict frame count の true/false 挙動が明示されている
- Hann/Hamming 係数式が実装されている
- stream identity が output topic に更新される
- invalid sample を clamp せず破棄する
- diagnostics に設定値、last frame count、counter が含まれる
- colcon pytest registration が存在する

## Integration / Launch

`test/integration` と `test/launch` は将来の ROS2 graph test 用の配置を確保する。現時点では `.gitkeep` のみを置く。
