# fa_time_stretch 仕様書

## 1. 目的

`fa_time_stretch` は `FLOAT32LE` interleaved `AudioFrame` の pitch を保ったまま duration を ratio 指定で変更する temporal processing node である。

## 2. 入力

- message: `fa_interfaces/msg/AudioFrame`
- encoding: `FLOAT32LE`
- bit depth: `32`
- layout: `interleaved`
- sample rate / channels: startup config と一致

## 3. 出力

出力は入力と同じ sample rate、channels、encoding、bit depth、layout を保持する。
出力 frame count は `stretch.ratio` と input frame count から計算される expected range と一致する必要がある。

## 4. 責務外

- device / file / network I/O
- resample
- latency compensation / jitter buffer
- pitch shifting
- VAD / ASR / embedding
- unsupported frame length の hidden padding / truncate

## 5. fail closed 条件

- stretch ratio が未指定または不正
- frame length / hop length / window 不正
- input frame metadata mismatch
- non-finite sample
- output frame count が expected range と一致しない
- output が finite/normalized contract を満たさない
