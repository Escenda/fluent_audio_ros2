# fa_asr アルゴリズム詳細説明書

1. `AudioFrame` を mono float samples に変換する。
2. VAD active 中の samples を buffer に追加する。
3. VAD end または timeout で buffer を finalize する。
4. backend に samples と sample rate を渡す。
5. transcript を `AsrResult` として publish する。

Python version / venv が異なる ASR engine は node 内 import ではなく external worker / process / container backend として扱います。
