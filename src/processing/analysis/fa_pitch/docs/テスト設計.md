# fa_pitch テスト設計

## 方針

`fa_pitch` は AI node ではないため、テストは model inference ではなく pitch measurement contract を対象にする。

## Test IDs

| ID | 観点 |
| --- | --- |
| `FA-PITCH-SPEC-001` | package が `src/processing/analysis` にあり、README / docs が AI node ではないことを明記する |
| `FA-PITCH-SPEC-002` | backend source が ROS2 / `fa_interfaces` を import しない |
| `FA-PITCH-SPEC-003` | `FLOAT32LE` mono normalized input だけを扱う |
| `FA-PITCH-SPEC-004` | backend output が finite で `frame_count` と同じ長さの pitch sequence を持つ |
| `FA-PITCH-SPEC-005` | default config は `backend.name=internal_autocorrelation` を明示する |
| `FA-PITCH-SPEC-006` | ROS parameter は default で補完せず、YAML/launch から明示されない場合は起動失敗する |
| `FA-PITCH-SPEC-007` | input `stream_id` が `input_topic` と一致しない frame は publish しない |
| `FA-PITCH-SPEC-008` | sample count が `n_fft + N * hop_length` に一致しない場合は truncate せず drop する |
| `FA-PITCH-SPEC-009` | silence は frequency zero、confidence zero、voiced false になる |
| `FA-PITCH-SPEC-010` | sample system config は 20ms 16kHz mic chunk と整合する `feature.n_fft=320` を明示する |
