# fa_vad アルゴリズム詳細説明書

1. `AudioFrame` を float mono samples に変換する。
2. 必要な場合は backend target sample rate へ resample する。
3. float samples を PCM16 bytes へ変換する。
4. backend `update()` に PCM16 bytes を渡す。
5. probability / is_speech / start / end を `VadState` として publish する。
6. speech state が変化した場合のみ Bool topic を publish する。

Resampling は backend input adaptation であり、source node の format negotiation ではありません。
