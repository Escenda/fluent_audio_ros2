# fa_kws アルゴリズム詳細説明書

1. `AudioFrame` を mono float samples に変換する。
2. 最新の `VadState.probability` を読む。
3. `VadState` が未受信、または `vad.max_age_ms` より古い場合は frame を処理せず skip する。
4. backend `process()` に samples, sample rate, VAD probability, timestamp を渡す。
5. detection があれば `WakeWordResult` を publish する。

VAD probability は gating signal です。VAD 推定は `fa_vad` に分離します。
sherpa-onnx provider は `backend.execution_provider` で明示し、空値や未対応 provider は backend 生成前に reject します。
sherpa-onnx KWS の current C API result は confidence score を返さないため、backend は binary detection score `1.0` を明示的に返します。
