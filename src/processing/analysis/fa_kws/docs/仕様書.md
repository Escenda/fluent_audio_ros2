# fa_kws 仕様書

## 責務

`fa_kws` は audio frame と VAD state から wake word を検出する analysis node です。backend を明示選択し、wake word event を publish します。

## 非責務

- VAD 推定そのもの
- ASR
- turn detection
- microphone source selection
- model fallback

## Input / Output

| 種別 | 名前 | 型 |
| --- | --- | --- |
| Sub | `audio_topic` | `fa_interfaces/msg/AudioFrame` |
| Sub | `vad_topic` | `fa_interfaces/msg/VadState` |
| Pub | `output_topic` | `fa_interfaces/msg/WakeWordResult` |

## Backend Contract

- `backend.name` は必須。
- backend は ROS2 topic/message を知らない。
- backend input は mono float samples, source sample rate, VAD probability, timestamp。
- backend output は detected flag, keyword, score。

## Fail Closed

- unknown backend
- model path missing
- tokens/keywords file missing
- invalid audio frame

## Test IDs

- `FA-KWS-SPEC-001`: unknown backend で起動失敗
- `FA-KWS-SPEC-002`: missing model file で起動失敗
- `FA-KWS-SPEC-003`: VAD gate 未満で backend result を publish しない
- `FA-KWS-SPEC-004`: detection を WakeWordResult に変換する
