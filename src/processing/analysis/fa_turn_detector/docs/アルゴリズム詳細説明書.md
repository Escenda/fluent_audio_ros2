# fa_turn_detector アルゴリズム詳細説明書

1. `AudioFrame` を mono float samples に変換する。
2. rolling buffer に samples を追加する。
3. VAD state と turn context を保持する。
4. 必要 sample 数が揃ったら backend に window を渡す。
5. probability が threshold を超えた場合に `TurnEnd` を publish する。

Turn context は ASR / dialogue 側との相関に使います。turn detector は会話状態を直接変更しません。
