# fa_source_separation 仕様書

## 1. 目的

`fa_source_separation` は audio mixture から明示 target source を分離し、source ごとの `AudioFrame` stream を生成する spatial processing node である。
neural backend は external worker / process / container 境界に分ける。

## 2. 入力/出力

入力は config と一致する `AudioFrame` とする。
出力は target source ごとの `AudioFrame` topic とし、source identity と target label を明示する。

## 3. 責務外

- beamforming
- speaker embedding / diarization
- ASR
- routing mixer
- resample / bit-depth conversion
- unknown source count の推測補完

## 4. fail closed 条件

- backend 未指定
- target source list 欠損
- model path / model id 欠損
- input metadata mismatch
- backend output count が target source list と一致しない
- output が finite/normalized contract を満たさない
