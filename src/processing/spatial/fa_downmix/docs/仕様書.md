# fa_downmix 仕様書

## 目的

`fa_downmix` は、FLOAT32LE / 32bit / interleaved の `fa_interfaces/msg/AudioFrame` に対して、明示的に許可された N-channel to mono/stereo downmix だけを行う。

## 仕様 ID

| ID | 内容 |
| --- | --- |
| `FA-DOWNMIX-SPEC-001` | `fa_downmix` は `FLOAT32LE` / 32 bit / interleaved `AudioFrame` に対して明示許可された N-channel to mono/stereo downmix だけを行う spatial processing node である。 |
| `FA-DOWNMIX-SPEC-002` | upmix、channel duplicate、resampling、sample format conversion、gain、limiter、pan、filtering、denoise、device I/O を行わない。 |
| `FA-DOWNMIX-SPEC-003` | `input_topic` / `output_topic` と `input_stream_id` / `output.stream_id` は別契約であり、topic identity と stream identity の衝突を起動時に拒否する。 |
| `FA-DOWNMIX-SPEC-004` | expected input channels、output channels、mode、expected format、QoS、diagnostics parameter は required で、不正値は fail closed する。 |
| `FA-DOWNMIX-SPEC-005` | `average_to_mono` と `pair_average_to_stereo` だけを許可し、`output.channels >= expected.input_channels` は downmix ではないため拒否する。 |
| `FA-DOWNMIX-SPEC-006` | runtime frame は source / stream identity、format、channel、data alignment、finite normalized input / output sample を検証する。 |
| `FA-DOWNMIX-SPEC-007` | 出力 metadata は入力を基準にし、`stream_id`、`channels`、`data` だけを更新する。 |
| `FA-DOWNMIX-SPEC-008` | invalid sample は clamp / normalize で補正せず drop する。 |
| `FA-DOWNMIX-SPEC-009` | diagnostics は topic、stream id、mode、expected format、output channels、frame counter を publish する。 |
| `FA-DOWNMIX-SPEC-010` | launch file は `node_name` / `config_file` を明示引数とし、package-local default config を推測しない。 |
| `FA-DOWNMIX-SPEC-011` | pytest contract は `colcon test` から実行され、`ament_lint_auto` dependency を package manifest に明示する。 |

## 起動時検証

以下を満たさない設定は起動失敗とする。

- `input_topic` と `output_topic` が空でない
- `input_stream_id` と `output.stream_id` が空でない
- `input_stream_id` / `output.stream_id` が raw / resolved ROS topic と衝突しない
- `input_stream_id` と `output.stream_id` が同一でない
- `expected.sample_rate > 0`
- `expected.input_channels > 0`
- `output.channels > 0`
- `expected.encoding == FLOAT32LE`
- `expected.bit_depth == 32`
- `expected.layout == interleaved`
- `mode == average_to_mono` かつ `expected.input_channels > output.channels` かつ `output.channels == 1`
- `mode == pair_average_to_stereo` かつ `expected.input_channels > output.channels` かつ `output.channels == 2` かつ `expected.input_channels` が偶数
- `qos.depth > 0`
- `diagnostics.publish_period_ms > 0`
- `diagnostics.qos.depth > 0`
- `diagnostics.qos.reliable` は bool

`output.channels >= expected.input_channels` は downmix ではないため許可しない。

## Runtime frame 検証

以下を満たさない frame は warning を出して drop する。

- `source_id` と `stream_id` が空でない
- `stream_id == input_stream_id`
- `encoding == FLOAT32LE`
- `bit_depth == 32`
- `layout == interleaved`
- `sample_rate == expected.sample_rate`
- `channels == expected.input_channels`
- `data` が空でない
- `data.size()` が `expected.input_channels * sizeof(float)` で割り切れる
- すべての入力 sample が finite かつ `[-1.0, 1.0]` 内
- すべての出力 sample が finite かつ `[-1.0, 1.0]` 内

## Metadata 更新

出力 frame は入力 frame を基準にし、以下だけを更新する。

- `stream_id = output.stream_id`
- `channels = output.channels`
- `data = downmixed bytes`

`header`、`source_id`、`sample_rate`、`bit_depth`、`encoding`、`layout`、`epoch` は保持する。

## 非対象

upmix、channel duplicate、resampling、sample format conversion、gain、limiter、pan、filtering、denoise、device I/O は別 package の責務とする。
