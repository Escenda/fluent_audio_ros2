# fa_downmix アルゴリズム詳細説明書

## 入力表現

入力は FLOAT32LE / interleaved の `AudioFrame.data` とする。1 sample は little-endian の 4 byte float として読む。

## `average_to_mono`

入力 channel 数は 2 以上、出力 channel 数は 1 とする。各 frame の channel sample `c0..cN-1` に対して、出力は全 channel の算術平均とする。

```text
in:  [c0, c1, c2, c3]
out: [(c0 + c1 + c2 + c3) / 4]
```

## `pair_average_to_stereo`

入力 channel 数は偶数 4 以上、出力 channel 数は 2 とする。channel を interleaved order の L/R pair とみなし、偶数 index channel を left 群、奇数 index channel を right 群として平均する。

```text
in:  [l0, r0, l1, r1, l2, r2]
out: [(l0 + l1 + l2) / 3, (r0 + r1 + r2) / 3]
```

奇数 channel 入力、2ch 入力の 2ch 出力、1ch 入力の 1ch 出力は downmix ではないため起動時に reject する。

## 範囲検証

入力 sample または出力 sample が finite でない、または normalized range `[-1.0, 1.0]` の外にある場合、その runtime frame 全体を drop する。clamp、normalize、zero fill は行わない。

## 非対象処理

本 package は downmix 以外の意味を変更しない。upmix、channel duplicate、sample format conversion、resampling、gain、limiter、pan、noise gate、filtering、denoise、device I/O は別 package の責務とする。
