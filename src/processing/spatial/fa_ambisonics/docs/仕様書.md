# fa_ambisonics 仕様書

## 1. 目的

`fa_ambisonics` は ambisonic channel representation の order / normalization / coordinate convention を明示して変換する spatial processing node である。

## 2. 入力/出力

入力と出力は `AudioFrame` とし、sample rate、encoding、bit depth、layout は config と一致する必要がある。
channel count は ambisonic order に従って検証する。

## 3. 責務外

- device / file / network I/O
- binaural rendering
- beamforming
- source separation
- resample / bit-depth conversion
- channel convention の自動推測

## 4. fail closed 条件

- ambisonic order / normalization / channel ordering 未指定
- input channel count mismatch
- unsupported transform
- non-finite sample
- output が finite/normalized contract を満たさない
