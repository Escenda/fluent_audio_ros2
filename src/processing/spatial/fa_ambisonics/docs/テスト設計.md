# fa_ambisonics テスト設計

## Unit Test

- `FA-AMBISONICS-SPEC-001`: package 化前でも README、仕様書、アルゴリズム詳細説明書、テスト設計、backend docs が存在する。
- `FA-AMBISONICS-SPEC-002`: `package.xml` と `CMakeLists.txt` を package 化前に置かない。
- `FA-AMBISONICS-SPEC-003`: 仕様書が channel convention の自動推測を禁止する。
- `FA-AMBISONICS-SPEC-004`: backend docs が ROS2 topic/message dependency を禁止する。

## Integration Test

ROS2 package 化後に、known transform matrix fixture で channel order と output metadata を確認する。

## Launch Test

ROS2 package 化後に、missing params file、unsupported order、channel count mismatch で起動失敗することを確認する。
