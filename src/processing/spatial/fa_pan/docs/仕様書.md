# fa_pan 仕様書

## 目的

`fa_pan` は FluentAudio の stereo pan processing package である。FLOAT32LE interleaved stereo `AudioFrame` に constant-power pan を適用し、音像配置後の stereo frame を publish する。

## Scope

対象:

- `fa_interfaces/msg/AudioFrame` の購読と publish
- `pan.position` による constant-power stereo pan
- 起動時 config validation
- runtime frame validation
- diagnostics publish

対象外:

- device I/O
- resampling
- sample format conversion
- channel-count conversion
- limiter / filter / denoise
- RMS / peak / VAD 等の解析 metadata 生成

## ROS2 Interface

| 種別 | 名前 | 型 | 内容 |
| --- | --- | --- | --- |
| subscribe | `input_topic` | `fa_interfaces/msg/AudioFrame` | pan 対象 stereo frame を運ぶ ROS topic |
| publish | `output_topic` | `fa_interfaces/msg/AudioFrame` | pan 後の stereo frame を運ぶ ROS topic |
| publish | `diagnostics` | `diagnostic_msgs/msg/DiagnosticArray` | pan position、gain、counter |

## Parameters

| Parameter | Default | Constraint |
| --- | --- | --- |
| `input_topic` | `fa_pan/input` | non-empty |
| `output_topic` | `fa_pan/output` | non-empty |
| `input_stream_id` | `audio/channel_converted/mic` | non-empty、raw/resolved `input_topic` / `output_topic` と不一致、`output.stream_id` と不一致 |
| `output.stream_id` | `audio/panned/mic` | non-empty、raw/resolved `input_topic` / `output_topic` と不一致、`input_stream_id` と不一致 |
| `pan.position` | `0.0` | finite, `-1.0 <= value <= 1.0` |
| `expected.sample_rate` | `16000` | `> 0` |
| `expected.channels` | `2` | exactly `2` |
| `expected.encoding` | `FLOAT32LE` | exactly `FLOAT32LE` |
| `expected.bit_depth` | `32` | exactly `32` |
| `expected.layout` | `interleaved` | exactly `interleaved` |
| `qos.depth` | `10` | `> 0` |
| `qos.reliable` | `false` | boolean |
| `diagnostics.publish_period_ms` | `1000` | `> 0` |
| `diagnostics.qos.depth` | `10` | `> 0` |
| `diagnostics.qos.reliable` | `true` | bool |

Invalid config は起動時に `std::runtime_error` で fail closed する。

## AudioFrame Contract

Runtime frame は以下を満たす必要がある。

- `source_id` が non-empty
- `stream_id` が non-empty
- `stream_id == input_stream_id`
- `sample_rate == expected.sample_rate`
- `channels == 2`
- `encoding == FLOAT32LE`
- `bit_depth == 32`
- `layout == interleaved`
- `data` が non-empty
- `data.size()` が `2 * sizeof(float)` の倍数
- 全入力 sample が finite normalized `[-1, 1]`
- 全出力 sample が finite normalized `[-1, 1]`

違反した runtime frame は warning を出して drop し、publish しない。clamp、normalize、default 補完は行わない。

## Output Preservation

出力 frame は入力 frame を基準に以下のみ更新する。

- `stream_id = output.stream_id`
- `data = pan 後の FLOAT32LE interleaved bytes`

`header`, `source_id`, `sample_rate`, `channels`, `bit_depth`, `encoding`, `layout`, `epoch` は保持する。
