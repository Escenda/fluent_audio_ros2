# fa_binaural 仕様書

## 1. 目的

`fa_binaural` は mono/stereo/multichannel または ambisonic stream を、明示 HRTF profile に基づいて headphone-oriented binaural stream へ rendering する spatial processing node である。

## 2. 入力/出力

入力は config で指定した `input_stream_id` と channel layout に一致する `AudioFrame` とする。
出力は 2ch `FLOAT32LE` interleaved `AudioFrame` とし、`source_id` を保持し `stream_id` を `output.stream_id` に更新する。
`input_topic` / `output_topic` は ROS 搬送路であり、`AudioFrame.stream_id` として扱わない。

## 3. 責務外

- headphone / speaker device を開くこと
- ambisonic order transform
- source separation
- resample / bit-depth conversion
- missing HRTF profile の fallback

## 4. fail closed 条件

- HRTF profile 欠損
- `input_stream_id` / `output.stream_id` の欠損、topic 名との衝突、または相互衝突
- input layout / channel count mismatch
- unsupported renderer
- non-finite sample
- output が finite/normalized contract を満たさない
