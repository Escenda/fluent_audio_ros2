# fa_interleave テスト設計

## Contract test

`test/unit/test_fa_interleave_audio_frame_contract.py` は、package の config、source、layout、colcon pytest registration を検査する。

## 検証項目

- default config が 16000Hz / FLOAT32LE / 32bit / 2ch / `interleaved` -> `planar` を明示する
- device I/O、resampling、sample format conversion、bit-depth conversion、channel count conversion、gain、limiter、filtering の責務を持たない
- 起動時に未対応 layout / format combination を reject する
- runtime frame が `source_id`、`stream_id`、format、layout、channel、data size を満たさない場合 warning + drop する
- `stream_id` を `output_topic` に、`layout` を `output.layout` に更新し、他 metadata を保持する
- interleaved -> planar と planar -> interleaved の byte reorder 実装証跡がある
- `ament_cmake_pytest` で contract test が colcon から実行される

## 実行コマンド

```bash
env PYTHONDONTWRITEBYTECODE=1 PYTEST_DISABLE_PLUGIN_AUTOLOAD=1 python3 -m pytest -p no:cacheprovider src/processing/format/fa_interleave/test/unit -q
```
