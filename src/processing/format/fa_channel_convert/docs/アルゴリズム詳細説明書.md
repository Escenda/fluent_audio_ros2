# fa_channel_convert アルゴリズム詳細説明書

## 入力表現

入力は FLOAT32LE / interleaved の `AudioFrame.data` とする。1 sample は little-endian の 4 byte float として読む。

## `mono_to_stereo_duplicate`

入力 channel 数は 1、出力 channel 数は 2 とする。各 mono sample `x` に対して、出力は `[x, x]` とする。

```text
in:  [m0, m1, m2]
out: [m0, m0, m1, m1, m2, m2]
```

## `stereo_to_mono_average`

入力 channel 数は 2、出力 channel 数は 1 とする。各 frame の left/right sample `l`, `r` に対して、出力は `(l + r) * 0.5` とする。

```text
in:  [l0, r0, l1, r1]
out: [(l0 + r0) / 2, (l1 + r1) / 2]
```

## 範囲検証

入力 sample または出力 sample が finite でない、または normalized range `[-1.0, 1.0]` の外にある場合、その runtime frame 全体を drop する。clamp、normalize、zero fill は行わない。

## 非対象処理

本 package は channel count 以外の意味を変更しない。sample format conversion、resampling、gain、limiter、noise gate、filtering、denoise、device I/O は別 package の責務とする。
