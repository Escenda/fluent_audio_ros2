# fa_encode テスト設計

## Unit Test

- `FA-ENCODE-SPEC-001`: config は input AudioFrame encoding / bit depth / sample rate / channels / layout、codec/container/bitrate、input/output stream identity を明示する。
- `FA-ENCODE-SPEC-002`: node validation は unsupported input AudioFrame capability、codec/container/backend protocol を fail closed で拒否する。
- `FA-ENCODE-SPEC-003`: 仕様書が `fa_out` に encode を隠さない責務分離を明記する。
- `FA-ENCODE-SPEC-004`: 仕様書が missing codec / backend / input format を fail closed として扱う。
- `FA-ENCODE-SPEC-005`: external worker / fake backend protocol は declared input contract と一致する PCM だけを encoded payload へ変換する。
- `FA-ENCODE-SPEC-006`: external command timeout、empty output、oversized output を error として扱う。
- `FA-ENCODE-SPEC-007`: `input_stream_id` / `output.stream_id` は必須で、ROS topic identity と同値にしない。

## Integration Test

fixture PCM `AudioFrame` を explicit external command backend に渡し、`stream_id == input_stream_id` かつ declared output contract 一致時だけ `stream_id = output.stream_id` の `EncodedAudioChunk` が publish されることを確認する。

## Launch Test

missing params file、missing backend、missing codec、missing command executable で起動失敗することを確認する。

## 仕様とテストケース対応

### 実装済み

- `FA-ENCODE-TC-001` -> `FA-ENCODE-SPEC-001`, `FA-ENCODE-SPEC-002`: config validation rejects missing input capability, codec/container, and unsupported input `AudioFrame` metadata.
- `FA-ENCODE-TC-002` -> `FA-ENCODE-SPEC-003`, `FA-ENCODE-SPEC-004`: launch fail-closed tests reject missing backend, missing codec, missing input format, and missing command.
- `FA-ENCODE-TC-003` -> `FA-ENCODE-SPEC-005`: fake backend protocol test accepts only PCM matching the declared input contract.
- `FA-ENCODE-TC-004` -> `FA-ENCODE-SPEC-006`: backend API test verifies timeout, empty output, oversized output, and encoded metadata mismatch are errors.
- `FA-ENCODE-TC-005` -> `FA-ENCODE-SPEC-007`: ROS graph / file pipeline tests publish encoded chunks only when input stream and input contract match.
- `FA-ENCODE-TC-006` -> `FA-ENCODE-SPEC-004`: launch fail-closed test covers invalid params file and backend startup failure.
