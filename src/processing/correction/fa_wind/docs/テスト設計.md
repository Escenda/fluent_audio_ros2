# fa_wind テスト設計

## Unit Test

- `FA-WIND-SPEC-001`: package 化前でも README、仕様書、アルゴリズム詳細説明書、テスト設計、backend docs が存在する。
- `FA-WIND-SPEC-002`: `package.xml` と `CMakeLists.txt` を package 化前に置かない。
- `FA-WIND-SPEC-003`: 仕様書が high-pass / gate fallback を禁止する。
- `FA-WIND-SPEC-004`: backend docs が ROS2 topic/message dependency を禁止する。

## Integration Test

ROS2 package 化後に、synthetic wind burst fixture を使って suppression と non-wind preservation を確認する。

## Launch Test

ROS2 package 化後に、missing params file、missing backend、invalid detector config で起動失敗することを確認する。
