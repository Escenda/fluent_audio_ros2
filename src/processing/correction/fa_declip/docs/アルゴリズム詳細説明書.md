# fa_declip アルゴリズム詳細説明書

1. startup config で clipping threshold、reconstruction window、expected format を検証する。
2. input `AudioFrame` の identity と format metadata を検証する。
3. backend で clipped region を検出する。
4. clipped region の前後 context から reconstruction を行う。
5. finite/normalized contract を検証し、契約を満たす場合だけ publish する。

clipping が解消できない場合に limiter や normalize で隠さない。
reconstruction confidence が threshold 未満の場合は drop または明示 error result とする。

## Backend Boundary

backend は clipped region detection と waveform reconstruction のみを担当し、ROS2 topic/message を知らない。
