# fa_debreath アルゴリズム詳細説明書

1. startup config で detection band、threshold、attenuation、expected format を検証する。
2. input `AudioFrame` の identity と format metadata を検証する。
3. backend で breath-like band energy と temporal envelope を評価する。
4. 明示 threshold を満たす範囲だけ attenuation curve を適用する。
5. finite/normalized contract を検証し、契約を満たす場合だけ publish する。

breath detection が不確かな場合に無音化しない。fail closed として drop または unchanged result の明示 status を返す。

## Backend Boundary

backend は breath candidate detection と attenuation のみを担当し、ROS2 topic/message を知らない。
