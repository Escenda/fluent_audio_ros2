# fa_debreath 仕様書

## 1. 目的

`fa_debreath` は `FLOAT32LE` interleaved `AudioFrame` から breath noise を抑制する correction processing node である。
息継ぎ音を完全に削除するのではなく、明示 config で指定した検出・抑制範囲に限定して attenuation を行う。

## 2. 入力/出力

入力と出力は同じ sample rate、channels、encoding、bit depth、layout を持つ `AudioFrame` である。
`source_id` は保持し、入力 `stream_id` は `input_stream_id` と一致する必要がある。
出力 `stream_id` は `output.stream_id` に更新する。`input_topic` / `output_topic`
は ROS 搬送路であり、`AudioFrame.stream_id` として扱わない。

## 3. 責務外

- device / file / network I/O
- resample / channel conversion / bit-depth conversion
- VAD / ASR / embedding
- loudness normalize / compressor
- breath 区間欠損時の推測補完

## 4. fail closed 条件

- backend 未指定
- detection band / threshold / attenuation が不正
- input metadata mismatch
- `input_stream_id` / `output.stream_id` の欠損、topic 名との衝突、または相互衝突
- non-finite sample
- output が finite/normalized contract を満たさない
