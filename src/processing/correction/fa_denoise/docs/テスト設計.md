# fa_denoise テスト設計

## unit test

- backend name validation。
- missing model path の起動失敗。
- block size と overlap-add の境界条件。
- invalid sample rate / channel / bit depth の drop。

## integration test

- `fa_resample -> fa_denoise` の topic wiring。
- DTLN model を使った short fixture の inference。

## launch test

- `config/default.yaml` で model が存在する環境では起動できる。
- ONNX Runtime がない build 環境では configure に失敗する。
- unknown backend で起動失敗する。

## failure test

- model file missing。
- unsupported frame format。
- inference exception。

## latency test

block 単位の inference latency、queueing、diagnostics counter を測る。

## real device smoke test

実 microphone から `fa_resample -> fa_denoise` を起動し、output frame の sample rate / channel / RMS / drop counter を確認する。
