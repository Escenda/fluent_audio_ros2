# fa_aec_nn アルゴリズム詳細説明書

## 処理手順

```text
AudioFrame
  ↓
validate expected format
  ↓
dispatch to backend
  ↓
publish processed frame
```

## passthrough backend

現在の `passthrough` backend は input frame をそのまま output topic へ publish する。これは未実装 NN を成功扱いにするものではなく、pipeline wiring と downstream contract の検証用 backend である。

## frame / buffer 前提

- sample rate は 16kHz。
- channels は `expected_channels` と一致すること。
- data は PCM frame boundary に一致すること。

## known limitations

- 実際の neural AEC model は未接続。
- model backend を追加する場合、runtime dependency と model path validation を `docs/backends/` と launch config に明記する。
