# fa_aec_linear アルゴリズム詳細説明書

## 処理手順

```text
reference AudioFrame
  ↓
validate and cache latest reference

mic AudioFrame
  ↓
validate
  ↓
if enabled and fresh reference exists:
      decode mic/ref to float32
      subtract cancel_gain * ref
      encode to original bit depth
  else:
      publish mic according to explicit enabled / timeout semantics
```

## cancellation

同じ sample layout の mic / ref を前提に、sample ごとに次を計算する。

```text
out[i] = mic[i] - cancel_gain * ref[i]
```

出力時は bit depth に合わせて clamp する。

## frame / buffer 前提

- sample rate は 16kHz。
- bit depth は 16 or 32。
- mic/ref は同一 channel 数であること。
- reference は `ref_timeout_ms` 内であること。

## known limitations

- adaptive filter ではない。
- echo path delay 推定をしない。
- reference と mic の frame alignment がずれると効果は限定的。
