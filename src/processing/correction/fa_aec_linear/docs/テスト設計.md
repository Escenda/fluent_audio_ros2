# fa_aec_linear テスト設計

## unit test

- matching mic/ref frame の sample subtraction。
- stale reference は mic pass-through ではなく drop。
- reference format mismatch は mic pass-through ではなく drop。
- mic/reference decode failure は mic pass-through ではなく drop。
- invalid sample rate / channel / encoding / bit depth の drop。
- `PCM32LE/32` を float32 として解釈せず drop。
- mic/reference sample count mismatch は短い側へ truncate せず drop。
- output range overflow は clamp せず drop。
- source code に `std::clamp`、saturating cast、hidden limiter が存在しない。

## integration test

- `fa_resample` の mic/ref output から `fa_aec_linear` へ接続する。
- `enabled=false` では mic pass-through せず drop する。

## launch test

- `config/default.yaml` で起動できる。
- empty topic / invalid timeout / sample rate 不一致で起動失敗する。

## failure test

- reference 欠落。
- frame size mismatch。
- bit depth unsupported。
- output range overflow。

## latency test

mic callback から output publish までの処理時間を測る。

## real device smoke test

speaker playback 中の microphone input を reference とともに通し、diagnostics の in/out/drop counter を確認する。
