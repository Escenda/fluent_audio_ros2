## integration test

- `fa_resample` の mic/ref output から `fa_aec_linear` へ接続する。
- `enabled=false` では mic pass-through せず drop する。

## launch test

- `config/default.yaml` で起動できる。
- empty topic / empty stream identity / stream identity と topic 名の衝突 / invalid timeout / sample rate 不一致 / invalid expected channel 数 / invalid expected encoding で起動失敗する。

## failure test

- reference 欠落。
- reference が mic より未来。
- reference が `ref_timeout_ms` を超えて古い。
- frame size mismatch。
- bit depth unsupported。
- output range overflow。

## latency test

mic callback から output publish までの処理時間を測る。

## real device smoke test

speaker playback 中の microphone input を reference とともに通し、diagnostics の in/out/drop counter を確認する。
