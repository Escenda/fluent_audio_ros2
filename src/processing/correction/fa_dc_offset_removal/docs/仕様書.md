# fa_dc_offset_removal 仕様書

## 目的

`fa_dc_offset_removal` は、`FLOAT32LE` interleaved `AudioFrame` stream から frame 内のチャンネル別平均値を差し引き、DC offset 成分を除去する correction processing node である。

## スコープ

### 対象

- `fa_interfaces/msg/AudioFrame` の購読と publish
- `FLOAT32LE/32/interleaved` の契約検証
- frame 単位のチャンネル別平均減算
- diagnostics への config と counters の publish

### 非対象

- device I/O
- resampling
- sample format conversion
- gain
- limiter
- noise gate
- high-pass filter
- notch filter
- normalize / clamp

## Topic

| 方向 | パラメータ | 型 | 既定値 |
| --- | --- | --- | --- |
| subscribe | `input_topic` | `fa_interfaces/msg/AudioFrame` | `audio/sample_format/mic` |
| publish | `output_topic` | `fa_interfaces/msg/AudioFrame` | `audio/dc_offset_removed/mic` |
| publish | `diagnostics` | `diagnostic_msgs/msg/DiagnosticArray` | `diagnostics` |

## 設定

| パラメータ | 既定値 | 制約 |
| --- | --- | --- |
| `expected.sample_rate` | `16000` | `> 0` |
| `expected.channels` | `1` | `> 0` |
| `expected.encoding` | `FLOAT32LE` | `FLOAT32LE` のみ |
| `expected.bit_depth` | `32` | `32` のみ |
| `expected.layout` | `interleaved` | `interleaved` のみ |
| `qos.depth` | `10` | `> 0` |
| `qos.reliable` | `false` | bool |
| `diagnostics.publish_period_ms` | `1000` | `> 0` |

起動時設定が制約を満たさない場合は例外で起動失敗する。暗黙の補正、既定値への差し替え、互換 alias は持たない。

## Runtime Frame Contract

入力 frame は以下を満たす必要がある。

- `source_id` が空でない
- `stream_id` が空でない
- `stream_id == input_topic`
- `sample_rate == expected.sample_rate`
- `channels == expected.channels`
- `encoding == expected.encoding`
- `bit_depth == expected.bit_depth`
- `layout == expected.layout`
- `data` が空でない
- `data.size()` が `channels * sizeof(float)` で割り切れる
- 全入力 sample が finite

違反した frame は warning を出して drop する。publish は行わない。

## 出力

出力 frame は入力 frame の以下を保持する。

- `header`
- `source_id`
- `sample_rate`
- `channels`
- `bit_depth`
- `encoding`
- `layout`
- `epoch`

出力 frame は以下を更新する。

- `stream_id = output_topic`
- `data = DC offset removal 後の FLOAT32LE byte列`

出力 sample が finite でない場合、その frame は drop する。

## Diagnostics

`diagnostics` に以下を publish する。

- `input_topic`
- `output_topic`
- `expected_sample_rate`
- `expected_channels`
- `frames_in`
- `frames_out`
- `frames_dropped`
- `backend.name=internal_frame_mean`
