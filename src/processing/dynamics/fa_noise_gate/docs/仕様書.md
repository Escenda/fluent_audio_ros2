# fa_noise_gate 仕様書

## 1. 目的

`fa_noise_gate` は FluentAudio の `AudioFrame` ストリームに対して、正規化済み
`FLOAT32LE` サンプルの小振幅成分だけを指定ゲインで減衰する dynamics processing node である。

## 2. 非目的

この package は以下を行わない。

- device I/O
- resampling
- sample format conversion
- channel count conversion
- compressor / limiter / normalize
- filtering / denoise

## 3. ノード

| 項目 | 値 |
| --- | --- |
| package | `fa_noise_gate` |
| executable | `fa_noise_gate_node` |
| node name | `fa_noise_gate` |

## 4. Topic

| 種別 | parameter | default |
| --- | --- | --- |
| subscribe | `input_topic` | `audio/dc_offset_removed/mic` |
| publish | `output_topic` | `audio/noise_gated/mic` |

## 5. 起動時パラメータ

| parameter | default | 条件 |
| --- | ---: | --- |
| `gate.threshold_linear` | `0.02` | finite, `0.0 <= value <= 1.0` |
| `gate.closed_gain_linear` | `0.0` | finite, `0.0 <= value <= 1.0` |
| `expected.sample_rate` | `16000` | `> 0` |
| `expected.channels` | `1` | `> 0` |
| `expected.encoding` | `FLOAT32LE` | `FLOAT32LE` のみ |
| `expected.bit_depth` | `32` | `32` のみ |
| `expected.layout` | `interleaved` | `interleaved` のみ |
| `qos.depth` | `10` | `> 0` |
| `qos.reliable` | `false` | bool |
| `diagnostics.publish_period_ms` | `1000` | `> 0` |

不正な起動時パラメータは fallback せず、例外で起動を失敗させる。

## 6. 入力フレーム契約

入力 `AudioFrame` は以下を満たす必要がある。

- `source_id` が非空
- `stream_id` が非空、かつ `input_topic` と一致
- `sample_rate` が `expected.sample_rate` と一致
- `channels` が `expected.channels` と一致
- `encoding == FLOAT32LE`
- `bit_depth == 32`
- `layout == interleaved`
- `data` が非空
- `data.size()` が `channels * sizeof(float)` で割り切れる
- 全サンプルが finite
- 全サンプルが `[-1.0, 1.0]` の正規化範囲内

実行時に契約違反を検出した frame は publish せず、警告ログを出して drop する。
値の補完、clamp、normalize、別形式としての解釈は行わない。

## 7. 出力フレーム契約

出力 `AudioFrame` は入力から以下を preserve する。

- header
- source_id
- sample_rate
- channels
- bit_depth
- encoding
- layout
- epoch

`stream_id` は `output_topic` に更新し、`data` は noise gate 適用後の `FLOAT32LE`
interleaved bytes に更新する。

## 8. Diagnostics

`diagnostics` topic に `DiagnosticArray` を publish する。status name は
`fa_noise_gate` とし、設定値と counters を含める。

- `gate_threshold_linear`
- `gate_closed_gain_linear`
- `expected_sample_rate`
- `expected_channels`
- `frames_in`
- `frames_out`
- `frames_dropped`
- `samples_gated`
- `output_topic`
