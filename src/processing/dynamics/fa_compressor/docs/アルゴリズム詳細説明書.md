# fa_compressor アルゴリズム詳細説明書

## 1. 概要

本 package は static curve のみを適用する compressor である。attack / release envelope、lookahead、RMS detector、peak hold、limiter は持たない。

## 2. 前提

- 入力は `FLOAT32LE` の interleaved PCM bytes。
- 各 sample は `float` として読み出す。
- 入力 sample は finite かつ `[-1.0, 1.0]` の範囲内でなければならない。

## 3. 圧縮式

入力 sample を `x`、振幅を `a = abs(x)`、threshold を `t`、ratio を `r`、makeup gain を `g` とする。

```text
if a <= t:
  compressed_abs = a
else:
  compressed_abs = t + (a - t) / r

y = sign(x) * compressed_abs * g
```

`y` が finite でない、または `[-1.0, 1.0]` を超える場合は frame 全体を drop する。sample 単位の clamp、limit、zero 埋めは行わない。

## 4. 圧縮カウンタ

`a > t` の sample を圧縮対象 sample と数え、frame publish 成功時に `samples_compressed` に加算する。output 範囲超過などで frame が drop された場合、その frame の圧縮数は加算しない。

## 5. metadata

入力 frame をコピーしたうえで `stream_id` と `data` のみ更新する。`header`、`source_id`、format metadata、`epoch` は変えない。
