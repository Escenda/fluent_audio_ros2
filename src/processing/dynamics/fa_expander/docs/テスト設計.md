# テスト設計

## 単体契約テスト

`test/unit/test_fa_expander_audio_frame_contract.py` はソース、設定、CMake、package manifest を検査し、以下を確認する。

- default config が FLOAT32LE interleaved の契約を宣言している
- 起動時設定検証が不正値を fail closed する
- 実行時フレーム検証が source_id、stream_id、format、layout、data size を確認する
- 出力 stream_id だけを `output_topic` に更新し、他のフレーム identity を保持する
- アルゴリズムが `threshold * pow(abs(x) / threshold, ratio)` を使う
- clamp やゼロ化を行わない
- diagnostics に設定値とカウンタを出す
- colcon から pytest が登録される

## 実行コマンド

```bash
env PYTHONDONTWRITEBYTECODE=1 PYTEST_DISABLE_PLUGIN_AUTOLOAD=1 python3 -m pytest -p no:cacheprovider src/processing/dynamics/fa_expander/test/unit -q
```
