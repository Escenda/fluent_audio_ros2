# fa_agc アルゴリズム詳細説明書

## 1. 入力

入力は `FLOAT32LE` interleaved の `AudioFrame.data` である。`channels = C`、総 sample 数を `N` とすると、`N` は `C` で割り切れる。frame 数は `F = N / C` である。

## 2. 検証

処理前に `AudioFrame` metadata と data size を検証する。さらに各 sample を `float` として読み、finite かつ normalized `[-1.0, 1.0]` であることを確認する。

検証失敗時はその frame を drop し、出力を publish しない。

## 3. Frame RMS 算出

全 sample の square mean から RMS を求める。

```text
frame_rms = sqrt(sum(sample[i] * sample[i]) / N) for i in 0..N-1
```

`frame_rms` が finite でない場合は frame を drop する。

## 4. Target gain

通常 frame の target gain は以下で求める。

```text
raw_target_gain = agc.target_rms / frame_rms
target_gain = min(max(raw_target_gain, agc.min_gain), agc.max_gain)
```

`frame_rms == 0.0` の場合、`raw_target_gain` は無限大へ向かうため、`target_gain = agc.max_gain` とする。入力 sample の検証は完了済みであり、zero sample は valid silence として扱う。

## 5. Attack / release smoothing

起動時の `current_gain` は `1.0` である。

gain を下げる場合は `agc.attack_ms`、gain を上げる場合は `agc.release_ms` を使い、frame duration から smoothing coefficient を求める。

```text
frame_seconds = (N / channels) / sample_rate
alpha = 1.0 - exp(-frame_seconds / time_constant_seconds)
candidate_gain = current_gain + alpha * (target_gain - current_gain)
```

`candidate_gain` は `min_gain` から `max_gain` の範囲に制限する。

## 6. 変換

全 sample に `candidate_gain` を掛ける。

```text
output[i] = input[i] * candidate_gain
```

全 output sample は finite かつ normalized `[-1.0, 1.0]` である必要がある。範囲外になる場合は frame を drop し、candidate gain を current gain として確定しない。clamp は行わない。

## 7. 出力 metadata

出力 `AudioFrame` は入力をコピーしてから `stream_id` と `data` のみ更新する。`header`、`source_id`、`sample_rate`、`channels`、`bit_depth`、`encoding`、`layout`、`epoch` は保持する。

## 8. Diagnostics

処理状態として AGC parameter、`current_gain`、`last_frame_rms`、`last_target_gain`、入力/出力/drop counter、gain 増減 counter を publish する。
