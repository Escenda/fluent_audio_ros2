# fa_gain テスト設計

## Unit

- default config が `FLOAT32LE` / 32bit / interleaved を要求する。
- source code に `std::clamp`、clip、normalize、resample、channel conversion が存在しない。
- `validateFrame` が source identity、layout、encoding、bit depth、sample rate、channels、data size を検証する。
- `applyGain` が `source_id` を保持し、`stream_id` を `output_topic` に更新する。
- CMake が pytest を colcon test に登録する。

## Integration

- `fa_resample -> fa_gain -> fa_vad` の topic wiring を確認する。
- gain 後に範囲外になる frame が publish されず、drop counter が増えることを確認する。

## Launch

- `ros2 launch fa_gain fa_gain.launch.py` が `config/default.yaml` で起動する。

## Fixtures

- 正規化済み `FLOAT32LE` mono 16kHz frame。
- gain 後に範囲外になる frame。
