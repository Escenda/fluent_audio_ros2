# fa_patchbay テスト設計

## 仕様 ID

| ID | 仕様 |
| --- | --- |
| `FA-PATCHBAY-SPEC-001` | default config key と node default name は `fa_patchbay` |
| `FA-PATCHBAY-SPEC-002` | `input_topics`、`input_stream_ids`、`output_topics`、`output_stream_ids` は同じ長さの parallel array |
| `FA-PATCHBAY-SPEC-003` | route tuple は unique で、input/output topic identity と topic/stream identity は衝突しない |
| `FA-PATCHBAY-SPEC-004` | expected format は `PCM16LE/16`, `PCM32LE/32`, `FLOAT32LE/32`, `interleaved` のみ |
| `FA-PATCHBAY-SPEC-005` | runtime frame は source_id、configured input stream identity、format、data alignment を検証する |
| `FA-PATCHBAY-SPEC-006` | valid frame は route ごとに copy され、変更 field は `stream_id` のみ |
| `FA-PATCHBAY-SPEC-007` | invalid frame は drop され、counter が増える |
| `FA-PATCHBAY-SPEC-008` | diagnostics は route_count、unique_input_count、input_topic.N、input_stream_id.N、resolved_input_topic.N、output_topic.N、output_stream_id.N、resolved_output_topic.N、frames_in、frames_dropped、copies_out を含む |
| `FA-PATCHBAY-SPEC-009` | package layout は FluentAudio processing package 標準に従う |
