# fa_voice_conversion 仕様書

## 1. 目的

`fa_voice_conversion` は入力 speech `AudioFrame` の内容や timing contract を保ちながら、明示 target voice condition に基づいて voice characteristics を変換する generation processing node である。

## 2. 入力/出力

入力は config と一致する speech `AudioFrame` とする。
出力は target voice condition に従う `AudioFrame` とし、sample rate、channels、encoding、layout は output config と一致する必要がある。

## 3. 責務外

- TTS
- speaker embedding extraction/generation
- ASR
- dialogue policy
- target voice 欠損時の fallback

## 4. fail closed 条件

- backend 未指定
- model path / model id 欠損
- target voice condition 欠損
- input metadata mismatch
- output が finite/normalized contract を満たさない
