# fa_neural_codec 仕様書

## 1. 目的

`fa_neural_codec` は waveform `AudioFrame` を neural codec token / latent representation へ変換し、必要に応じて逆変換する generation processing node である。
通常 codec encode/decode は `fa_encode` / `fa_decode` の責務であり、この node では扱わない。

## 2. 入力/出力

mode は `encode` または `decode` として明示する。
`encode` は `AudioFrame` から codec representation を出力し、`decode` は codec representation から `AudioFrame` を出力する。

## 3. 責務外

- PCM codec / transport codec encode
- TTS / vocoder
- ASR / embedding
- device / file / network I/O
- missing model の fallback

## 4. fail closed 条件

- mode 未指定
- backend 未指定
- model path / model id 欠損
- payload encoding mismatch
- output representation schema mismatch
- output が finite/normalized contract を満たさない
