# fa_neural_codec アルゴリズム詳細説明書

1. startup config で mode、backend、model path/model id、payload encoding を検証する。
2. mode に応じて input payload schema を検証する。
3. backend に canonical waveform または neural codec representation を渡す。
4. backend output の schema、sample rate、channels、finite/normalized contract を検証する。
5. 契約を満たす場合だけ publish する。

transport codec への fallback は行わない。
Python version / venv が異なる backend は external worker / process / container 境界にする。

## Backend Boundary

backend は neural codec model の encode/decode のみを担当し、ROS2 topic/message を知らない。
