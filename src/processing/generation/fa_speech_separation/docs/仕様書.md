# fa_speech_separation 仕様書

## 1. 目的

`fa_speech_separation` は混合 speech `AudioFrame` から複数 speech source を分離し、source ごとの `AudioFrame` stream を生成する generation processing node である。
source count / target labels は config で明示する。

## 2. 入力/出力

入力は config と一致する `AudioFrame` とする。
出力は target speech source ごとの `AudioFrame` topic とする。

## 3. 責務外

- speaker diarization
- speaker embedding / identification
- ASR
- routing mixer
- unknown source count の自動推測

## 4. fail closed 条件

- backend 未指定
- source count / target labels 欠損
- model path / model id 欠損
- backend output count mismatch
- output が finite/normalized contract を満たさない
