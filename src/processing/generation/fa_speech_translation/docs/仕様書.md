# fa_speech_translation 仕様書

## 1. 目的

`fa_speech_translation` は入力 speech `AudioFrame` を target language の speech `AudioFrame` へ変換する generation processing node である。
ASR、translation、TTS を内部に隠す場合でも backend contract と model ownership を明示し、fallback で別経路に切り替えない。

## 2. 入力/出力

入力は source language と format metadata が config と一致する `AudioFrame` とする。
出力は target language speech の `AudioFrame` とし、target language を metadata / diagnostics に明示する。

## 3. 責務外

- dialogue policy
- standalone ASR result publish
- standalone TTS
- MCP / robot command execution
- missing model fallback

## 4. fail closed 条件

- backend 未指定
- source / target language 欠損
- model path / model id 欠損
- input metadata mismatch
- output が finite/normalized contract を満たさない
