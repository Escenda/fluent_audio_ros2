# fa_neural_vocoder 仕様書

## 1. 目的

`fa_neural_vocoder` は acoustic feature representation から `AudioFrame` waveform を生成する generation processing node である。
text frontend や linguistic model は責務外とし、feature-to-waveform contract に限定する。

## 2. 入力/出力

入力は mel spectrogram などの feature message / payload とし、feature schema、hop length、sample rate、speaker condition を明示する。
出力は `FLOAT32LE` interleaved `AudioFrame` とする。

## 3. 責務外

- text-to-speech frontend
- ASR / embedding
- resample / bit-depth conversion
- speaker output / sink adapter
- missing model fallback

## 4. fail closed 条件

- backend 未指定
- model path / model id 欠損
- feature schema mismatch
- conditioning 欠損
- output が finite/normalized contract を満たさない
