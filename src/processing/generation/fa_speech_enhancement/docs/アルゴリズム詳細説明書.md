# fa_speech_enhancement アルゴリズム詳細説明書

1. startup config で backend、model path/model id、expected format を検証する。
2. input `AudioFrame` の identity と format metadata を検証する。
3. backend に canonical audio tensor を渡す。
4. backend output の frame count、sample rate、channels、finite/normalized contract を検証する。
5. 契約を満たす場合だけ publish する。

enhancement 失敗時に original input を pass-through しない。
Python version / venv が異なる backend は external worker / process / container 境界にする。

## Backend Boundary

backend は speech enhancement inference のみを担当し、ROS2 topic/message を知らない。
