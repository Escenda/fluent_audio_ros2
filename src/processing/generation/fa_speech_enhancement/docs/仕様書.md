# fa_speech_enhancement 仕様書

## 1. 目的

`fa_speech_enhancement` は noisy speech `AudioFrame` から enhanced speech `AudioFrame` を生成する generation processing node である。
neural enhancement backend は external worker / process / container 境界に分ける。

## 2. 入力/出力

入力と出力は config と一致する `FLOAT32LE` interleaved `AudioFrame` とする。
`source_id` は保持し、`stream_id` は `output_topic` に更新する。

## 3. 責務外

- VAD / ASR / embedding
- AEC / dereverb / beamforming
- resample / channel conversion / bit-depth conversion
- device / file / network I/O
- missing model fallback

## 4. fail closed 条件

- backend 未指定
- model path / model id 欠損
- input metadata mismatch
- backend output frame count mismatch
- output が finite/normalized contract を満たさない
