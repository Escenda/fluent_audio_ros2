# fa_super_resolution 仕様書

## 1. 目的

`fa_super_resolution` は低 sample rate / 低帯域の `AudioFrame` から高帯域 waveform を生成する generation processing node である。
sample rate の単純変換は `fa_resample` の責務であり、この node は bandwidth extension model に限定する。

## 2. 入力/出力

入力は config と一致する low-resolution `AudioFrame` とする。
出力は target sample rate / target bandwidth の `AudioFrame` とし、output format は config で明示する。

## 3. 責務外

- simple resample
- EQ / high-pass / low-pass
- denoise / enhancement
- speaker output
- missing model fallback

## 4. fail closed 条件

- backend 未指定
- model path / model id 欠損
- target sample rate / bandwidth 欠損
- input metadata mismatch
- output が finite/normalized contract を満たさない
