# fa_super_resolution テスト設計

## Unit Test

- `FA-SUPER-RESOLUTION-SPEC-001`: package 化前でも README、仕様書、アルゴリズム詳細説明書、テスト設計、backend docs が存在する。
- `FA-SUPER-RESOLUTION-SPEC-002`: `package.xml` と `CMakeLists.txt` を package 化前に置かない。
- `FA-SUPER-RESOLUTION-SPEC-003`: 仕様書が resample fallback を禁止する。
- `FA-SUPER-RESOLUTION-SPEC-004`: backend docs が external worker 境界を明記する。

## Integration Test

ROS2 package 化後に、fake super-resolution backend で target sample rate と output metadata を確認する。

## Launch Test

ROS2 package 化後に、missing params file、missing backend、missing model、missing target rate で起動失敗することを確認する。
