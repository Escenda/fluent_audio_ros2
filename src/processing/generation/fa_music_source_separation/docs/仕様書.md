# fa_music_source_separation 仕様書

## 1. 目的

`fa_music_source_separation` は音楽 mixture から明示 target stem を分離し、stem ごとの `AudioFrame` stream を生成する generation processing node である。
neural backend は external worker / process / container 境界に分ける。

## 2. 入力/出力

入力は config と一致する `AudioFrame` とする。
出力は target stem ごとの `AudioFrame` topic とし、`vocals`、`drums`、`bass`、`accompaniment` などの stem label を明示する。

## 3. 責務外

- routing mixer
- speaker diarization / speaker embedding
- ASR
- device / file / network I/O
- target stem list の自動推測

## 4. fail closed 条件

- backend 未指定
- target stem list 欠損
- model path / model id 欠損
- backend output count が target stem list と一致しない
- output が finite/normalized contract を満たさない
