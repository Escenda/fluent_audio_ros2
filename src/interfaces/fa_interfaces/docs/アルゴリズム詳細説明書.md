# fa_interfaces アルゴリズム詳細説明書

## 処理手順

`fa_interfaces` はアルゴリズムを実行しない。生成される C++ / Python interface 型が FluentAudio graph の data contract になる。

## 型設計

`AudioFrame` は interleaved PCM chunk を前提にする。実際の encoding、sample rate、channel 数、bit depth は message field に明示し、受信 node が期待値と照合する。

```text
AudioFrame
  header
  source_id
  stream_id
  encoding
  sample_rate
  channels
  bit_depth
  layout
  data
  epoch
```

`EncodedAudioChunk` は codec encode 済み chunk を前提にする。`AudioFrame.encoding` は PCM sample representation を表す field なので、Opus / AAC / FLAC などの codec payload を `AudioFrame` に詰め込まない。

```text
EncodedAudioChunk
  header
  source_id
  stream_id
  codec
  container
  payload_format
  sample_rate
  channels
  sequence
  media_time_ns
  duration_ns
  epoch
  data
```

RMS / peak / loudness / VAD / KWS / ASR / TD は `AudioFrame` を直接書き換えるのではなく、それぞれ専用 message を publish する。turn 単位の関連付けは `TurnContext` と各 result message の session / turn field で行う。

`VadState` は VAD 判定に使った `AudioFrame` の `source_id` / `stream_id` をそのまま保持する。これにより KWS / ASR / Turn Detector は別 source / stream の VAD state を audio buffer に混ぜずに reject できる。

`PlaybackControl` は `fa_out` の output lifecycle 制御専用 service である。`stop` は active epoch を進め、queue 内 frame と旧 epoch frame を破棄する。`pause` / `resume` は backend への新規 chunk write を停止 / 再開する。`PlaybackControl` は resample、decode、gain、routing、TTS などの音声処理要求を受け付けない。

## frame / buffer 前提

- `AudioFrame.data` は `channels * bytes_per_sample` の倍数であること。
- `source_id` と `stream_id` は空でないこと。
- `VadState.source_id` / `stream_id` は対応する `AudioFrame` identity と一致すること。
- `layout` は現在 `interleaved` のみを正とすること。
- `encoding` は受信 node の contract と一致していること。
- `sample_rate=0`、`channels=0`、空 data は有効な音声 frame として扱わない。
- `EncodedAudioChunk.codec` / `container` / `payload_format` は空でないこと。
- `EncodedAudioChunk.duration_ns=0`、空 data は有効な encoded chunk として扱わない。
- `EncodedAudioChunk.media_time_ns` は `stream_id` と `epoch` 内で monotonic に扱うこと。

## known limitations

- schema evolution は後方互換を残さず、呼び出し元も同時に更新する。
- planar audio は現在の `AudioFrame` では直接表現しない。必要な場合は format package で明示的に変換する。
